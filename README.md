# Portfolio

## User Acceptance

```
AS AN employer
I WANT to view a potential employee's deployed portfolio of work samples
SO THAT I can review samples of their work and assess whether they're a good candidate for an open position

```

## Acceptance Criteria

```
I will meet this goal
WHEN I have a nav bar
THAT will navigate to the corrilating section on the page.
WHEN The user hovers over images those images are interactive.
WHEN they get to the contact section.
THOSE elements are active and are able to be used
```
